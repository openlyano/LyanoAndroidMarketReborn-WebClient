# Lyano Android Market: Reborn - Web Client

Android Studio source code for the Android Market: Reborn Web App client with JSbridge and ROOT Auto install.
Made in studio 3.6.
android 2.3+


important notes: 
- you can compile in studio 2.3.2 by setting gradle plugin from 3.0.0 to 2.3.2
- you can change the offline page in /assets/offline.html
